## 💄 Rawaa Beauty Store - SQL Data Analysis

A comprehensive SQL project focused on building and analyzing a database for a beauty brand (*Rawaa Store*). This project demonstrates database design, data integrity, and business intelligence reporting.

## ✨ Key Technical Skills Applied:
* *Relational Database Design:* Created a structured schema for Products, Customers, and Orders.
* *Data Integrity:* Implemented FOREIGN KEYS and ON DELETE RESTRICT to prevent accidental data loss.
* *Advanced Reporting:* Developed *SQL Views* for Customer Loyalty tracking and Sales Analysis.
* *Performance Optimization:* Used *Indexing* on product search fields for faster query execution.

## 🛠️ Tools Used:
* *MySQL Workbench*
* *SQL Language* (DDL, DML, DQL)
